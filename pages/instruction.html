ℹ️ AboutOm

BearStrength is built around a simple three-step workflow: build your exercise library, create workouts, and log your sessions. Everything else — history, progress data, and stats — is generated automatically.

BearStrength bygger på ett enkelt trestegsflöde: bygg ditt övningsbibliotek, skapa träningspass och logga dina pass. Allt annat — historik, progressionsdata och statistik — genereras automatiskt.

Step 1Steg 1

Build Your Exercise Library

Bygg ditt övningsbibliotek

✨ Create an Exercise (Wand / + Button)Skapa en övning (Trollstav / + Knapp)

Tap the wand icon or + button in the Exercises tab to add a custom exercise. Enter the exercise name and select a muscle group — that's it.

Tryck på trollstavsikonen eller + knappen i fliken Övningar för att lägga till en egen övning. Ange övningens namn och välj en muskelgrupp — det är allt.

📦 Import Default ExercisesImportera standardövningar

Don't want to build from scratch? Tap the wand icon and choose to import a pre-built library of common exercises, available in both:

Vill du inte börja från noll? Tryck på trollstavsikonen och välj att importera ett färdigt bibliotek med vanliga övningar, tillgängligt på båda:

  • English
  • Swedish
  • Engelska
  • Svenska
Tip: Duplicate detection is built in — importing won't add exercises you already have.
Tips: Dubblettdetektering är inbyggd — import lägger inte till övningar du redan har.

🗂️ Managing ExercisesHantera övningar

  • Exercises are listed alphabetically with their muscle group
  • Swipe left on any exercise to delete it
  • Exercises in active workouts are protected from accidental deletion
  • Övningar listas alfabetiskt med sin muskelgrupp
  • Svep åt vänster på en övning för att ta bort den
  • Övningar som ingår i aktiva träningspass är skyddade mot oavsiktlig borttagning
Step 2Steg 2

Create a Workout

Skapa ett träningspass

📋 What is a Workout?Vad är ett träningspass?

A workout is a saved plan — a named list of exercises you want to perform together. For example: "Push Day", "Leg Day", or "Full Body A".

Ett träningspass är en sparad plan — en namngiven lista med övningar du vill utföra tillsammans. Till exempel: "Pushdag", "Bendag" eller "Helkropp A".

➕ Creating a WorkoutSkapa ett träningspass

  1. Go to the Workouts tab and tap +
  2. Give the workout a name (e.g., "Push Day")
  3. Tap the workout to open it and add exercises
  4. Search by name or muscle group to find exercises quickly
  5. Tap an exercise to add it — a checkmark shows it's included
  1. Gå till fliken Träning och tryck på +
  2. Ge träningspasset ett namn (t.ex. "Pushdag")
  3. Tryck på träningspasset för att öppna det och lägga till övningar
  4. Sök på namn eller muskelgrupp för att hitta övningar snabbt
  5. Tryck på en övning för att lägga till den — en bockmarkering visar att den är inkluderad

↕️ Reordering & EditingÄndra ordning & redigera

  • Drag and drop exercises within a workout to set your preferred order
  • Swipe left on an exercise to remove it from the workout
  • When browsing exercises to add, you can see which other workouts already include them
  • Dra och släpp övningar i ett träningspass för att ange din föredragna ordning
  • Svep åt vänster på en övning för att ta bort den från träningspasset
  • När du bläddrar bland övningar att lägga till kan du se vilka andra träningspass som redan innehåller dem
Step 3Steg 3

Log a Workout

Logga ett träningspass

▶️ Starting a WorkoutStarta ett träningspass

  1. Go to the Start tab
  2. Pick a workout from the dropdown — you'll see a preview of the exercises
  3. Tap Start to begin
  1. Gå till fliken Start
  2. Välj ett träningspass från rullgardinsmenyn — du får en förhandsvisning av övningarna
  3. Tryck på Start för att börja
Tip: The app automatically records your start time so workout duration is tracked for you.
Tips: Appen registrerar automatiskt din starttid så att träningspassets längd spåras åt dig.

📝 Logging SetsLogga set

  1. Tap an exercise name to expand it
  2. Enter reps and weight (kg) for your set, then tap Add Set
  3. Repeat for each set — your total set count is shown at the bottom
  4. Tap Edit next to any logged set to correct a mistake, or swipe to delete it
  1. Tryck på ett övningsnamn för att expandera det
  2. Ange reps och vikt (kg) för ditt set, tryck sedan på Lägg till set
  3. Upprepa för varje set — ditt totala antal set visas längst ner
  4. Tryck på Redigera bredvid ett loggat set för att korrigera ett misstag, eller svep för att ta bort det

🔁 See Your Previous PerformanceSe din tidigare prestation

While in an active workout, each exercise shows the reps and weight from your last session for that same exercise — so you always know what to beat.

Under ett aktivt träningspass visar varje övning reps och vikt från ditt senaste pass för samma övning — så du alltid vet vad du ska slå.

💬 Exercise Notes / CommentsÖvningsanteckningar / Kommentarer

Below each exercise you can type a note — technique cues, how it felt, an injury reminder, anything. Notes from your previous session are shown automatically so you can refer back to them.

Under varje övning kan du skriva en anteckning — teknikpåminnelser, hur det kändes, en skadepåminnelse, vad som helst. Anteckningar från ditt föregående pass visas automatiskt så att du kan hänvisa till dem.

Note: Notes are stored per exercise per workout. They'll appear in History and be shown again next time you do the same workout.
Obs: Anteckningar lagras per övning per träningspass. De visas i Historik och visas igen nästa gång du gör samma träningspass.

✅ Finishing a WorkoutAvsluta ett träningspass

  1. Tap Finish Workout when done
  2. A summary screen shows duration, all exercises, every set, and your notes
  3. Tap Done to save and return to the home screen
  4. To discard a session, tap Cancel Workout — you'll be asked to confirm
  1. Tryck på Avsluta träning när du är klar
  2. En sammanfattningsskärm visar varaktighet, alla övningar, varje set och dina anteckningar
  3. Tryck på Klar för att spara och återgå till startskärmen
  4. För att avbryta ett pass, tryck på Avbryt träning — du ombeds bekräfta
Step 4Steg 4

View Your History

Visa din historik

📅 Workout HistoryTräningshistorik

The History tab shows all completed workouts, newest first. Each entry shows the date and total sets logged.

Fliken Historik visar alla avslutade träningspass, nyast först. Varje post visar datum och totalt antal loggade set.

  • Tap a workout to see the full breakdown — every exercise, every set, and any notes
  • Swipe left on a session to delete it
  • Use the delete-all option to clear the entire history
  • Tryck på ett träningspass för att se hela nedbrytningen — varje övning, varje set och eventuella anteckningar
  • Svep åt vänster på ett pass för att ta bort det
  • Använd alternativet ta bort allt för att rensa hela historiken

Pro Tips

Pro-tips

🚀 Getting Started FastKom igång snabbt

  • Import the default exercise library (English or Swedish) to skip manual entry
  • Create at least one workout before starting a session
  • You can add more exercises to a workout at any time — even between sessions
  • Importera standardövningsbiblioteket (engelska eller svenska) för att hoppa över manuell inmatning
  • Skapa minst ett träningspass innan du startar en session
  • Du kan lägga till fler övningar i ett träningspass när som helst — även mellan sessioner

⚖️ Weight InputViktinmatning

  • Weight is entered in kg
  • Decimal values are supported — use either a comma or a period (e.g., 22.5 or 22,5)
  • Vikt anges i kg
  • Decimalvärden stöds — använd antingen komma eller punkt (t.ex. 22,5 eller 22.5)

🔒 Data SafetyDatasäkerhet

  • Exercises that are part of a workout cannot be deleted until removed from it first
  • Cancelling a workout discards all data from that session — a confirmation prompt will always appear first
  • Deleting history or exercises requires a confirmation step
  • Övningar som är en del av ett träningspass kan inte tas bort förrän de tagits bort därifrån
  • Att avbryta ett träningspass raderar all data från det passet — en bekräftelseprompt visas alltid först
  • Att ta bort historik eller övningar kräver ett bekräftelsesteg

🌍 Language SupportSpråkstöd

  • Default exercises are available in both English and Swedish
  • You can import both sets — duplicates are filtered out automatically
  • Standardövningar finns på både engelska och svenska
  • Du kan importera båda uppsättningarna — dubbletter filtreras bort automatiskt
